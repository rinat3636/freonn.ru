# Freonn — инженерные системы под ключ

Freonn — инженерная компания из Москвы и Московской области. Проектируем, монтируем и обслуживаем системы вентиляции, кондиционирования, дымоудаления, отопления, холодоснабжения, водоснабжения, электроснабжения и пескоструйной обработки.

- **Сайт:** [https://freonn.ru](https://freonn.ru)
- **Регион:** Москва и Московская область
- **Опыт:** с 2011 года, более 1280 выполненных объектов
- **Гарантия:** 1 год на монтажные работы

## Услуги

- Монтаж и проектирование вентиляции
- Монтаж и проектирование кондиционирования
- Системы дымоудаления и противопожарной защиты
- Монтаж отопления и холодоснабжения
- Водоснабжение, электроснабжение, пескоструйная обработка
- Автоматизация и диспетчеризация инженерных систем
- Сервисное обслуживание

## Технологический стек

- React + TypeScript + Vite
- Tailwind CSS
- Express + Node.js
- SEO: server-side meta injection, JSON-LD, sitemap-index.xml
